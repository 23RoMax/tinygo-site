# TinyGo Web Site

This is the web site for TinyGo.

Built using Hugo:

http://gohugo.io/

and the Hugo "Learn" theme:

https://github.com/matcornic/hugo-theme-learn

## Deploy to GH Pages

- Commit all changes to master.
- Run `publish.sh` script.
